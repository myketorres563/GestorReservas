> Un sistema de gestión de reservas integral construido con React, TypeScript y Vite, diseñado para agilizar el proceso de reserva para empresas y particulares.


## Descripción
Gestion Reservas es un sistema de gestión de reservas de código abierto que permite a los usuarios gestionar las reservas de manera eficiente. El sistema está construido con una interfaz de usuario amigable, lo que facilita a las empresas y particulares realizar un seguimiento y gestión de sus reservas.

## Características
- 🎯 **Gestión de Reservas**: Gestione las reservas con facilidad, incluyendo la creación, edición y eliminación de reservas.
- ⚡ **Actualizaciones en Tiempo Real**: Reciba actualizaciones en tiempo real sobre el estado de las reservas, lo que le permite mantenerse informado y actualizado.
- 📊 **Informes y Análisis**: Genere informes y análisis para ayudarlo a tomar decisiones basadas en datos.
- 🔒 **Seguridad**: Construido con la seguridad en mente, lo que garantiza la protección y seguridad de sus datos.

## Pila de Tecnologías
- **Frontend**: React, TypeScript, CSS3, HTML5
- **Backend**: Node.js, Express.js (no incluido en el proyecto)
- **Herramienta de Construcción**: Vite
- **Administrador de Paquetes**: npm
- **Contenerización**: Docker

## Requisitos Previos
- Node.js (versión 16 o superior)
- npm (versión 8 o superior)
- Docker (opcional)

## Instalación
### 1. Instalar dependencias de raíz:
```bash
npm install
```

### 2. Instalar dependencias del backend:
```bash
cd backend
npm install
```

## Uso
### Modo de Desarrollo
```bash
npm run dev
```

### Modo de Producción
```bash
npm run build
npm run preview
```
